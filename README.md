# onurozansunger.github.io

Source code for [**onurozansunger.github.io**](https://onurozansunger.github.io) — my personal portfolio website.

## Stack

- HTML5 · CSS3 · vanilla JavaScript
- [Inter](https://rsms.me/inter/) + [JetBrains Mono](https://www.jetbrains.com/lp/mono/) (Google Fonts)
- Hosted on **GitHub Pages**

## Structure

| File | Purpose |
| --- | --- |
| `index.html` | Single-page site (Home, About, Experience, Projects, Contact) |
| `style.css`  | Styles, theme tokens, responsive layout, animations |
| `main.js`    | Mobile nav toggle, scroll-reveal animations |

## Local preview

```bash
python3 -m http.server 8000
# then visit http://localhost:8000
```
